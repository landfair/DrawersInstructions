Welcome to Drawers!

A powerful bookmarking and resource organization tool designed for educators, researchers, and knowledge workers who need more than traditional browser bookmarks.

What You'll Learn

Time required: About 5 minutes

What you'll need: Google Chrome browser

Step 1: Download the Extension

Download from Google Drive

  1. Visit the Google Drive folder: Click here to open
  2. Click the "Download" button at the top
  3. The folder will download as a .zip file to your Downloads folder
üí° Tip: The download might take a moment. You'll see a Drawers.zip file in your Downloads folder when it's complete.

Step 2: Extract the Files

Unzip the downloaded folder

On Mac:

  1. Go to your Downloads folder
  2. Double-click the Drawers.zip file
  3. A new Drawers folder will appear

On Windows:

  1. Go to your Downloads folder
  2. Right-click the Drawers.zip file
  3. Select "Extract All..."
  4. Click "Extract"
üí° Important: Move the extracted Drawers folder to your Desktop or another permanent location. Don't leave it in Downloads where you might accidentally delete it!

Step 3: Open Chrome Extensions

Access the Extensions page

There are two ways to get there:

Method 1 (Easiest):

  1. Open Google Chrome
  2. In the address bar, type: chrome://extensions/
  3. Press Enter

Method 2 (Menu):

  1. Click the three dots (‚ãÆ) in the top-right corner of Chrome
  2. Hover over "More tools"
  3. Click "Extensions"

Step 4: Enable Developer Mode

Turn on Developer Mode

  1. Look for the "Developer mode" toggle in the top-right corner of the Extensions page
  2. Click the toggle to turn it ON
  3. The toggle should turn blue when enabled
üí° Why Developer Mode? This allows you to load extensions from your computer instead of only from the Chrome Web Store. It's perfectly safe!
What you'll see: Once enabled, you'll see new buttons appear: "Load unpacked", "Pack extension", and "Update"

Step 5: Load the Extension

Install Drawers in Chrome

  1. Click the "Load unpacked" button (it appeared after enabling Developer mode)
  2. A file browser window will open
  3. Navigate to where you saved the Drawers folder (Desktop or wherever you moved it)
  4. Click on the Drawers folder to select it
  5. Click "Select Folder" (Mac) or "Select" (Windows)
‚ö†Ô∏è Important: Make sure you select the Drawers folder itself, not a file inside it. The folder should contain files like manifest.json, popup.html, and popup.js.

Step 6: Verify Installation

Check that Drawers is installed

You should see:

  1. The Drawers extension card appear in your extensions list
  2. The extension should show as "Enabled"
  3. A Drawers icon in your Chrome toolbar (top-right, near the address bar)

Can't see the icon?

  1. Click the puzzle piece icon (üß©) in your Chrome toolbar
  2. Find "Drawers" in the list
  3. Click the pin icon next to it to keep it visible
‚úÖ Success! If you can see the Drawers extension card on the extensions page, you've successfully installed it!

Step 7: Start Using Drawers

Create your first drawer

  1. Click the Drawers icon in your Chrome toolbar
  2. The Drawers side panel will open
  3. Click "+ Add Drawer"
  4. Name your drawer (e.g., "Research Papers", "Teaching Resources", "Project Links")
  5. Click "Create"

Add your first card

  1. Navigate to any webpage you want to save
  2. Open Drawers and select your drawer
  3. Click "+ Add Card"
  4. The card will auto-fill with the page's title, URL, and description
  5. Add tags if you want
  6. Click "Save"
